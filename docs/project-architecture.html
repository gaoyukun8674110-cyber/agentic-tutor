Boundary notes
架构关注点和风险
端口和 URL 默认值不一致backend/start.py 启动 8001，README 写 8000，apiClient.ts fallback 是 8001，frontend/.env.example 当前为 HTTPS 8000。生产部署需要统一 API base、CORS origin、反向代理和 HSTS 策略。
用户隔离主键混用业务表多处用 users.username 作为 user_id，LLM 凭证用 users.id。这能降低迁移成本，但后续如果重命名 username 会影响业务隔离语义。
RAG 索引是本地文件快照资料 chunk 入库后会重建 storage/materials/.vector-index.json。多实例部署时需要共享存储或迁移到数据库/向量库，否则实例间索引会分叉。
中文源码存在编码显示问题部分源码和文档在当前 PowerShell 输出中呈现 mojibake。功能不一定受影响，但维护体验和审阅质量会下降，建议统一 UTF-8 读写与编辑器设置。
